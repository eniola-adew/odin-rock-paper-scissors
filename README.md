# Rock Paper Scissors Game

A simple command-line implementation of the classic Rock, Paper, Scissors game where a human player competes against the computer over 5 rounds.

## Overview

This project implements a playable Rock, Paper, Scissors game in JavaScript. The player is prompted to enter their choice via a dialog box, and the computer makes a random selection. The game tracks scores for both the player and computer over 5 rounds and displays the final results.

## Features

- **Interactive Gameplay**: Players input their choice (rock, paper, or scissors) through a browser prompt
- **Random Computer Choices**: The computer opponent selects moves randomly from the three available options
- **Score Tracking**: Keeps track of wins and losses throughout the game
- **Console Output**: Displays results of each round and the final score

## How to Play

1. Open `index.html` in a web browser
2. When prompted, enter your choice: **rock**, **paper**, or **scissors**
3. The game will display the result of each round in the console
4. After 5 rounds, your final score and the computer's score will be displayed

## Game Rules

- **Rock** beats **Scissors**
- **Scissors** beats **Paper**
- **Paper** beats **Rock**
- If both choices are the same, the round is a tie (neither player scores)

## File Structure
